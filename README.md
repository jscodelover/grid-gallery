# GRID GALLERY

Using -
- pug
- sass
- glup (is a cross-platform, streaming task runner that lets developers automate many development tasks)
- glup-pug
- glup-sass

## Scripts

| Script             | Description                                          |
| ------------------ | ---------------------------------------------------- |
| `npm install`      | Installs all dependencies                            |
| `npm run gulp`     | compile pug and sass file into html and css file     |
